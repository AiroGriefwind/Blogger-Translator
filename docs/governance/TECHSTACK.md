# TECHSTACK

## 语言与运行时

- Python 3.11+
- Streamlit

## 核心库

- `requests` + `beautifulsoup4`：网页抓取与解析
- `python-docx`：文档生成与格式控制
- `google-cloud-storage` / `firebase-admin`：Firebase Storage
- `python-dotenv`：本地环境变量

## 模型服务

- SiliconFlow Chat Completions API
- 默认模型：`deepseek-r1`

## 配置密钥

- `SILICONFLOW_API_KEY`
- `SILICONFLOW_BASE_URL`
- `SILICONFLOW_MODEL`
- `FIREBASE_STORAGE_BUCKET`
- `GOOGLE_APPLICATION_CREDENTIALS`
