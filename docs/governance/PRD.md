# PRD

## 核心目标

用户在 Streamlit 输入巴士的报文章 URL 后，系统自动输出已翻译、已润色、已格式化的最终 `docx`。

## 功能范围

1. 抓取文章：标题、发布时间、作者、正文段落、图片 captions。
2. 首轮翻译：调用 SiliconFlow（默认 `deepseek-r1`）+ `Translate_Bot_Prompt`。
3. 人名核对清单：自动产出待检索问题列表。
4. 二轮润色：调用 `Revision_Bot_Prompt`，输出英中交替段落结构。
5. 长度控制：标题 > 12 词、Caption > 25 词时触发更短更有力改写。
6. 文档排版：按指定字号、加粗/斜体、双倍行距导出 `docx`。
7. 存储归档：使用 Firebase Storage 保存中间结果、产物、日志、译名映射。

## 必须达成指标

- 单 URL 处理流程可端到端执行成功。
- 输出文档满足排版规则。
- 每次运行都可在 Storage 找到完整归档。
- 失败步骤能记录错误日志并提示可读信息。
